## 🔍 6. THE SOCRATIC BRAIN (CORE)

This is your **multi-agent AI pipeline** that makes your tutor *pedagogical, not greedy*.

```
Input → Planner → Code Reasoner + RAG → Tutor → Verifier → Output
```

---

### 🔹 6.1 PLANNER

**Goal:**
Determine the *type of hint* required:

* Concept focus
* Bug category
* Difficulty tier
* Question tone

**Output Example**

```json
{
  "intent": "question",
  "focus": "loop logic",
  "severity": "medium"
}
```

---

### 🔹 6.2 RAG (RETRIEVAL)

**Purpose**
Help reasoners by giving real code examples:

* Common bugs
* Correct patterns
* Language idioms

**How it works**

1. Embed incoming code/error
2. Query vector database (hosted)
3. Return top-k similar snippets
4. Feed into AI prompts

**Example Providers**

* Hugging Face embeddings API
* OpenRouter model embeddings

---

### 🔹 6.3 CODE REASONER

**What it does**

* Deep analysis of bug
* Combines error + code context + RAG
* Outputs a *reasoning summary*

Not visible to user.

---

### 🔹 6.4 SOCRATIC TUTOR

**Model choices**
(use free providers)

Primary:
✔ `eth-nlped/TutorRL-7B` — Socratic behavior

Fallback / variation:
✔ `openrouter/free` — auto picks free model
✔ `eurecom-ds/phi-3-mini-4k-socratic` — strict Socratic trained

**Role**

* Generate multiple *hint candidates*
* ALWAYS ask a question
* NEVER give complete answer

Example prompt:

```text
You are a Socratic coding tutor.
Given: buggy code + error + reasoning summary
Ask one guiding question.
No direct solutions.
```

---

### 🔹 6.5 VERIFIER

**Goal**
Choose the *best Socratic hint*.

**Evaluation Criteria**
✔ No solution dumping
✔ Guides reasoning
✔ Targets bug
✔ Socratic style

**Implementation**
Prompt an LLM to score each candidate hint from 0–10.

Pick the highest scoring one.

## 🧠 8. GAME SCORING FORMULA

```
score = +1000
        - (hint_count × 50)
        - (time_seconds × 0.5)
```

**Interpretation**
✔ Hints are penalties
✔ Faster solves score better
✔ First correct wins + highest score
